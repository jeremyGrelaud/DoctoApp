DoctoApp

Purpose of our project

This app aims to help the elderly people managing their health. For that we will provide them an interactive list of their medication with timed notifications to avoid a poor medical treatment. The app will also allow them to see and manage their doctors appointement and lastly a tutor/guardian system will be implemented in the app. Therefore a person responsible of the elderly user would be alerted by sms in case of a forgotten dosing time.

Challeges & how to do it

One of the main challenges will be to manage the schedule of the customer. The schedule have to be easy to change for the user and we have to connect it to the guardian. So we have to create a simple and clear database with good interactions.
We choose to create this project in java, because we will need to use the object oriented method. As all the members of the group learned java in class this is a good choice.
